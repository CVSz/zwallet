# zWallet

## Overview
Multi-chain crypto wallet + swap platform.

## Stack
- Android (Kotlin)
- Backend (Node.js + TS)
- PostgreSQL + Redis
- Docker + Kubernetes

## Features
- Multi-chain wallet (EVM, Solana, BTC)
- Swap engine (DEX aggregation)
- Secure key management

## Run
```bash
docker-compose up
```

## Test
```bash
npm test
```
